Oefening: Hoogste Getal Berekenen

In deze oefening leer je hoe je met Math.max() het grootste getal van drie ingevoerde getallen kunt berekenen.

Voer de volgende code uit in je console:

let getal1 = parseInt(prompt("Geef getal in:"));
let getal2 = parseInt(prompt("Geef getal in:"));
let getal3 = parseInt(prompt("Geef getal in:"));
function hoogste(a, b, c) {
    return Math.max(a, b, c);
}
console.log(hoogste(getal1, getal2, getal3));

Tip: Open de console in je browser en voer drie getallen in wanneer daarom wordt gevraagd. De console zal het hoogste getal van de drie getoonde getallen weergeven.

Hoe werkt deze oefening?

Bij het uitvoeren van dit script wordt de gebruiker gevraagd om drie getallen in te voeren. De functie hoogste gebruikt vervolgens Math.max() om het grootste getal van deze drie te bepalen en retourneert het resultaat in de console.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomst te zien.
