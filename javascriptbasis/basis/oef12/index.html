Oefening: Gebruiker Begroeten
Voorbeeldcode

// Functie om een gebruiker te begroeten
function begroet(naam) {
    return `Hallo, ${naam}!`;
}

// Test de functie
console.log(begroet("Jan")); // Verwachte Output: "Hallo, Jan!"

Hoe werkt deze oefening?

In deze oefening definieer je een functie begroet die een naam als parameter neemt en een gepersonaliseerde begroeting retourneert. Klik op de Run knop om de console-uitvoer te bekijken in het vak hieronder.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
