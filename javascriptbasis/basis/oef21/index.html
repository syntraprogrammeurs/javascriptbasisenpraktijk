Oefening: Tel Woorden
Voorbeeldcode

// Functie om het aantal woorden in een zin te tellen
function telWoorden(zin) {
    return zin.split(" ").length; // Split de zin op spaties en tel het aantal woorden
}

// Test de functie
console.log(telWoorden("Dit is een voorbeeldzin."));  // Verwachte Output: 4

Hoe werkt deze oefening?

In deze oefening maak je een functie telWoorden die het aantal woorden in een zin telt door deze te splitsen op spaties. Klik op de Run knop om het resultaat voor de voorbeeldzin te zien.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
