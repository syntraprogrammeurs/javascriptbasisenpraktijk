Oefening: Dagen Tellen Tussen Twee Datums

In deze oefening leer je hoe je het aantal dagen tussen twee datums kunt berekenen. De functie dagenTussen neemt twee datums als parameters en retourneert het aantal dagen tussen deze datums.

Voer de volgende code uit in je console:

function dagenTussen(datum1, datum2) {
    const tijdverschil = Math.abs(Date.parse(datum2) - Date.parse(datum1));
    return tijdverschil / (1000 * 60 * 60 * 24);
}

console.log(dagenTussen('2024-11-04', '1973-11-08'));

Tip: Open de console in je browser en voer twee datums in om het aantal dagen tussen de datums te zien.

Hoe werkt deze oefening?

De functie dagenTussen berekent het verschil in milliseconden tussen de twee datums en deelt dit door het aantal milliseconden in een dag. Hierdoor krijg je het aantal dagen tussen de datums. Dit kan worden uitgebreid met een functie om jaren en maanden te berekenen.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomst te zien.
