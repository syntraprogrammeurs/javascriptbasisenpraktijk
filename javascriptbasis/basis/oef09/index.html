Oefening: Array Methodes Demonstratie

In deze oefening leer je hoe je verschillende array-methoden in JavaScript kunt gebruiken om arrays te manipuleren en te bewerken. Hieronder volgt een uitleg van elke methode:

push() Methode

De push() methode voegt een of meer elementen toe aan het einde van een array en retourneert de nieuwe lengte van de array.

pop() Methode

De pop() methode verwijdert het laatste element uit een array en retourneert dat element.

shift() Methode

De shift() methode verwijdert het eerste element uit een array en retourneert dat element. De resterende elementen verschuiven een index naar links.

unshift() Methode

De unshift() methode voegt een of meer elementen aan het begin van een array toe en retourneert de nieuwe lengte van de array.

forEach() Methode

De forEach() methode voert een opgegeven functie uit voor elk element in de array, maar retourneert geen nieuwe array. Hier wordt elke naam in de array namen gelogd naar de console.

map() Methode

De map() methode maakt een nieuwe array door een opgegeven functie uit te voeren op elk element in de array. In dit geval wordt elk getal verdubbeld.

filter() Methode

De filter() methode maakt een nieuwe array met alle elementen die voldoen aan de opgegeven conditie. Hier worden alleen even getallen gefilterd uit de array.

find() Methode

De find() methode retourneert het eerste element in de array dat voldoet aan de opgegeven conditie. In dit geval zoekt het naar het eerste getal dat groter is dan 2.

findIndex() Methode

De findIndex() methode retourneert de index van het eerste element dat voldoet aan de opgegeven conditie. Hier is het de index van het eerste getal dat groter is dan 2.

slice() Methode

De slice() methode maakt een nieuwe array met een gedeelte van de originele array. In dit geval worden de elementen van index 3 tot 6 uit de array gehaald.

splice() Methode

De splice() methode wijzigt de inhoud van een array door elementen te verwijderen of toe te voegen. Hier wordt een element op index 3 vervangen door "druiven".

reduce() Methode

De reduce() methode voert een reducer-functie uit op elk element in de array om uiteindelijk een enkel resultaat te verkrijgen. Hier wordt de som van alle getallen in de array berekend door ze te vermenigvuldigen.

Code Voorbeeld

Voer de volgende code uit in je console om de uitvoer van elke methode te zien:

let namen = ["Jan", "Piet", "Klaas"];

console.log(namen[1]);
console.log(namen.push("Fabio"));
console.log(namen[3]);
console.log(namen);
console.log(namen.push("Tom"));
console.log(namen);
console.log(namen.pop());
console.log(namen);
console.log(namen.shift());
console.log(namen);
console.log(namen.unshift("Jonas", "Dieter"));
console.log(namen);
console.log(namen.length);

namen.forEach(mijnNaam => console.log(mijnNaam));

let getallen = [1, 2, 3];
let verdubbeld = getallen.map(getal => getal * 2);
console.log(verdubbeld);

let numbers = [1, 2, 3, 4, 5];
let evenGetallen = numbers.filter(getal => getal % 2 === 0);
console.log(evenGetallen);

let nieuwenummers = [1, 2, 3, 4];
let eersteGroterDanTwee = nieuwenummers.find(getal => getal > 2);
console.log(eersteGroterDanTwee);
let index = nieuwenummers.findIndex(getal => getal > 2);
console.log(index);

let fruits = ["appel", "banaan", "peer", "kiwi", "kersen", "sinaasappel"];
let sommigeFruits = fruits.slice(3, 6);
console.log(sommigeFruits);

let tvfruits = fruits.splice(3, 1, "druiven");
console.log(tvfruits);
console.log(fruits);

let newnumbers = [1, 2, 3, 4];
let som = newnumbers.reduce((acc, num) => acc * num, 1);
console.log(som);

Hoe werkt deze oefening?

Deze code gebruikt verschillende JavaScript array-methoden om de mogelijkheden van array-manipulatie te demonstreren. Bekijk hoe elke methode werkt door het resultaat in de console te volgen.
