Oefening: Aantal Woorden Tellen

In deze oefening leer je hoe je met JavaScript het aantal woorden in een zin kunt tellen door gebruik te maken van de split-methode.

Voer de volgende code uit in je console:

let mijnZin = "Dit is een zin";
function telWoorden(zin) {
    return zin.split(" ").length;
}
console.log(telWoorden(mijnZin));

Tip: Open de console in je browser om het aantal woorden te zien dat door de functie wordt berekend.

Hoe werkt deze oefening?

Bij het uitvoeren van dit script splitst de functie telWoorden de zin op door te zoeken naar spaties. Het aantal verkregen delen is het aantal woorden in de zin, en dit wordt teruggegeven in de console.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomst te zien.
