Oefening: Klinkers Vervangen
Voorbeeldcode

// Functie om alle klinkers in een string te vervangen door het teken *
function vervangKlinkers(tekst) {
    return tekst.replace(/[aeiouAEIOU]/g, '*'); // Vervangt alle klinkers (zowel kleine als hoofdletters) door *
}

// Test de functie
console.log(vervangKlinkers("Hallo Wereld")); // Verwachte Output: "H*ll* W*r*ld"

Hoe werkt deze oefening?

In deze oefening maak je een functie vervangKlinkers die elke klinker in een tekst vervangt door het teken *. Klik op de Run knop om de output in het vak hieronder te zien.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
