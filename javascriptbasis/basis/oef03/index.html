Oefening: Even of Oneven

In deze oefening leer je hoe je een functie schrijft die controleert of een getal even is. De functie isEven geeft true terug als het getal even is, en false als het oneven is.

Voer de volgende code uit in je console:

let testGetal = parseInt(prompt("Geef een getal in"));
function isEven(getal) {
    return getal % 2 === 0;
}
console.log(isEven(testGetal));

Tip: Open de console in je browser en voer een getal in om het resultaat van de functie te bekijken.

Hoe werkt deze oefening?

Bij het uitvoeren van dit script wordt de gebruiker gevraagd om een getal in te voeren. De functie isEven controleert vervolgens of het getal even is door te berekenen of het deelbaar is door 2. Het resultaat (true of false) wordt weergegeven in de console.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomst te zien.
