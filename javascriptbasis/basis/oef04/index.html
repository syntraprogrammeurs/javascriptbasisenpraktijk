Oefening: Begroet Gebruikers

In deze oefening leer je hoe je een functie schrijft die een naam ontvangt en een persoonlijke begroeting retourneert. De functie begroet retourneert bijvoorbeeld "Hallo Tom!" als de ingevoerde naam "Tom" is.

Voer de volgende code uit in je console:

let mijnNaam = prompt("Geef je naam in:");
function begroet(naam) {
    return `Hallo ${naam}!`;
}
console.log(begroet(mijnNaam));

Tip: Open de console in je browser, voer je naam in wanneer daarom gevraagd wordt, en bekijk het resultaat.

Hoe werkt deze oefening?

Bij het uitvoeren van dit script wordt de gebruiker gevraagd om zijn of haar naam in te voeren. De functie begroet retourneert vervolgens een persoonlijke begroeting die in de console wordt weergegeven.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomst te zien.
