Oefening: Vermenigvuldigen van Twee Getallen

In deze oefening leer je hoe je een functie maakt om twee getallen met elkaar te vermenigvuldigen. De gebruiker voert twee getallen in, waarna het product wordt berekend en weergegeven in de console.

Voer de volgende code uit in je console:

let getala = parseInt(prompt("Geef getal in:"));
let getalb = parseInt(prompt("Geef getal in:"));
function vermenigvuldig(getal1, getal2) {
    return getal1 * getal2;
}
console.log(vermenigvuldig(getala, getalb));

Tip: Open de console in je browser, voer de getallen in als daarom gevraagd wordt, en bekijk het resultaat.

Hoe werkt deze oefening?

Bij het uitvoeren van dit script worden gebruikers gevraagd om twee getallen in te voeren. De functie vermenigvuldig() berekent het product van deze getallen. Het resultaat wordt getoond met console.log.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomst te zien.
