Oefening: Kwadrateren van een Getal
Voorbeeldcode

// Functie om een getal te kwadrateren
function vierkant(getal) {
    return getal * getal;
}

// Test de functie
console.log(vierkant(4)); // Verwachte Output: 16

Hoe werkt deze oefening?

In deze oefening definieer je een functie vierkant die een getal als parameter neemt en het kwadraat van dat getal retourneert. Klik op de Run knop om de console-uitvoer te bekijken in het vak hieronder.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
