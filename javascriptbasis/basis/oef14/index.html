Oefening: Even of Oneven Controleren
Voorbeeldcode

// Functie om te controleren of een getal even is
function isEven(getal) {
    return getal % 2 === 0;
}

// Test de functie
console.log(isEven(6)); // Verwachte Output: true
console.log(isEven(7)); // Verwachte Output: false

Hoe werkt deze oefening?

In deze oefening definieer je een functie isEven die controleert of een getal even is. Als het getal even is, retourneert de functie true, anders retourneert ze false. Klik op de Run knop om de output in het vak hieronder te zien.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen.
