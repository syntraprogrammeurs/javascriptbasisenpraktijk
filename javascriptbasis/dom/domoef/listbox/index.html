Oefening: Optie Overdracht Tussen Listboxen

Deze oefening demonstreert hoe je geselecteerde opties kunt overdragen van één listbox naar een andere met behulp van knoppen.

Voorbeeldcode
// Functie om geselecteerde opties te verplaatsen
function moveSelected(fromListbox, toListbox) {
    var selectedOptions = Array.from(fromListbox.selectedOptions); // Maak een array van de geselecteerde opties
    selectedOptions.forEach(function(option) {
        toListbox.appendChild(option); // Voeg de optie toe aan de doel-listbox
    });
}

// Event listeners voor de knoppen
toRightBtn.addEventListener('click', function() {
    moveSelected(listbox1, listbox2);
});

toLeftBtn.addEventListener('click', function() {
    moveSelected(listbox2, listbox1);
});

Hoe werkt deze oefening?

  • Selectie van opties: De gebruiker kan meerdere opties selecteren in de listbox.
  • Overdracht naar rechts of links: De geselecteerde opties worden verplaatst naar de andere listbox wanneer op de overeenkomstige knop wordt gedrukt.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomsten te volgen.
