Kleur van Tekst Veranderen bij Hover met JavaScript

Terug naar Home
Oefening: Tekstkleur veranderen bij hover
Voorbeeldcode
// JavaScript voor het veranderen van de tekstkleur bij hover
const tekstElement = document.getElementById("hoverTekst");

tekstElement.addEventListener("mouseenter", function() {
    tekstElement.style.color = "blue";
});

tekstElement.addEventListener("mouseleave", function() {
    tekstElement.style.color = "black";
});

Zweef met de muis over deze tekst om de kleur te veranderen.

Uitleg

In deze oefening leer je hoe je met JavaScript de kleur van een tekst kunt veranderen wanneer de gebruiker met de muis erover zweeft.

  • Doel: De tekstkleur veranderen naar blauw wanneer de gebruiker met de muis over de tekst zweeft.
  • Functie: Een JavaScript-functie die de tekstkleur verandert bij hover en terugzet wanneer de muis weggaat.
  • Interactie: Beweeg de muis over de tekst om de kleurverandering te zien.
