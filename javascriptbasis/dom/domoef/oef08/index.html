Toon Verborgen Paragraaf met JavaScript

Terug naar Home
Oefening: Verborgen paragraaf zichtbaar maken met een knop
Voorbeeldcode
// JavaScript voor het tonen van een verborgen paragraaf
document.getElementById("toonParagraafBtn").addEventListener("click", function() {
    const paragraaf = document.getElementById("verborgenParagraaf");
    paragraaf.style.display = "block";
});

Dit is een verborgen paragraaf die zichtbaar wordt na het klikken op de knop.

Uitleg

In deze oefening leer je hoe je met JavaScript een verborgen paragraaf zichtbaar kunt maken door op een knop te klikken.

  • Doel: Een verborgen paragraaf tonen wanneer de gebruiker op de knop klikt.
  • Functie: Een JavaScript-functie die de stijl van de paragraaf aanpast om deze zichtbaar te maken.
  • Interactie: Klik op de knop om de verborgen paragraaf te laten verschijnen.
