Oefening: Achtergrondkleur Wijzigen

In deze oefening leer je hoe je de achtergrondkleur van de pagina kunt wijzigen door te klikken op een knop. De knop activeert een JavaScript-functie die een willekeurige kleur selecteert uit een lijst van voorinstellingen.

Voorbeeldcode
const kleuren = [ "lightcoral", "lightseagreen", "lightsalmon", "lightpink", "lightgreen", "lightskyblue", "lightblue", "lightsteelblue", "lightyellow", "lightgoldenrodyellow", "lavender", "thistle", "plum", "violet", "aquamarine", "khaki", "peachpuff", "palegreen", "paleturquoise", "palevioletred" ];

function getRandomColor() {
    const randomIndex = Math.floor(Math.random() * kleuren.length);
    return kleuren[randomIndex];
}

let knop = document.getElementById("veranderKleur");
knop.addEventListener("click", function (){
    document.body.style.backgroundColor = getRandomColor();
});

Hoe werkt deze oefening?

  • Klik op de knop: Wanneer je op de knop "Klik mij" klikt, wordt de achtergrondkleur van de pagina gewijzigd naar een willekeurige kleur uit de lijst.
  • Gebruik van Math.random(): De functie getRandomColor kiest een willekeurige kleur door een index te genereren op basis van de lengte van de lijst.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de kleurverandering te volgen.
