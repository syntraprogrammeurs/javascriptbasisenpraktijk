Tekstinhoud Omwisselen met JavaScript

Terug naar Home
Oefening: Tekstinhoud van twee elementen omwisselen met een knop
Voorbeeldcode
// JavaScript voor het omwisselen van tekstinhoud
document.getElementById("omwisselTekstBtn").addEventListener("click", function() {
    const element1 = document.getElementById("element1");
    const element2 = document.getElementById("element2");
    const tempText = element1.textContent;
    element1.textContent = element2.textContent;
    element2.textContent = tempText;
});
Dit is de tekst van Element 1
Dit is de tekst van Element 2

Uitleg

In deze oefening leer je hoe je met JavaScript de tekstinhoud van twee elementen kunt omwisselen door op een knop te klikken.

  • Doel: De tekstinhoud van twee elementen omwisselen wanneer de gebruiker op de knop klikt.
  • Functie: De knop activeert een JavaScript-functie die de tekstinhoud van beide elementen omwisselt.
  • Interactie: Klik op de knop om de tekstinhoud van de twee elementen omgewisseld te zien worden.
