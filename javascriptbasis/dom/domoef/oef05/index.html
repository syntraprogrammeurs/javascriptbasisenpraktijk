Paragraaf Toevoegen met JavaScript

Terug naar Home
Oefening: Nieuwe paragraaf toevoegen met een knop
Voorbeeldcode
// JavaScript voor het toevoegen van een nieuwe paragraaf
document.getElementById("voegParagraafToeBtn").addEventListener("click", function() {
    const nieuweParagraaf = document.createElement("p");
    nieuweParagraaf.textContent = "Dit is een nieuw element";
    document.getElementById("paragraafContainer").appendChild(nieuweParagraaf);
});

Uitleg

In deze oefening leer je hoe je met JavaScript een nieuwe paragraaf toevoegt aan de pagina door op een knop te klikken.

  • Doel: Een nieuwe paragraaf toevoegen aan de container wanneer de gebruiker op de knop klikt.
  • Functie: De knop activeert een JavaScript-functie die een paragraaf aanmaakt en toevoegt.
  • Interactie: Klik op de knop om te zien hoe er dynamisch een nieuw element aan de pagina wordt toegevoegd.
