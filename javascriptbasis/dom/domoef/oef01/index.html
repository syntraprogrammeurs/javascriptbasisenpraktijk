Oefening: Achtergrondkleur Wijzigen

In deze oefening leer je hoe je de achtergrondkleur van de pagina kunt wijzigen door te klikken op een knop. De knop activeert een JavaScript-functie die een willekeurige kleur genereert met transparantie.

Voorbeeldcode
function getRandomRGBColorOpacity(){
    const r = Math.floor(Math.random() * 256);
    const g = Math.floor(Math.random() * 256);
    const b = Math.floor(Math.random() * 256);
    const a = Math.random().toFixed(1);
    return `rgba(${r}, ${g}, ${b}, ${a})`;
}

let knop = document.getElementById("veranderKleur");
knop.addEventListener("click", function (){
    document.body.style.backgroundColor = getRandomRGBColorOpacity();
});

Hoe werkt deze oefening?

  • Klik op de knop: Wanneer je op de knop "Wijzig Achtergrondkleur" klikt, wordt de achtergrondkleur van de pagina gewijzigd naar een willekeurige kleur.
  • Gebruik van Math.random(): De functie getRandomRGBColorOpacity maakt gebruik van Math.random() om willekeurige waarden te genereren voor de kleuren rood, groen en blauw (RGB) en voor de transparantie (Alpha).
  • Kleur in rgba-formaat: De gegenereerde kleur in rgba()-formaat zorgt voor een variabele transparantie, waardoor het kleureneffect dynamisch en doorzichtig wordt.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de kleurverandering te volgen.
