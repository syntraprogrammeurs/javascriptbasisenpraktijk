Huidige Tijdstip in Koptekst met JavaScript

Terug naar Home
Oefening: Huidige tijd weergeven en elke seconde bijwerken
Voorbeeldcode
// JavaScript voor het weergeven en bijwerken van de huidige tijd
function updateTijd() {
    const nu = new Date();
    const tijdString = nu.toLocaleTimeString();
    document.getElementById("tijdKoptekst").textContent = "Huidige tijd: " + tijdString;
}

setInterval(updateTijd, 1000); // Bijwerken elke seconde

Uitleg

In deze oefening leer je hoe je met JavaScript de huidige tijd in een koptekst kunt weergeven en deze elke seconde bijwerkt.

  • Doel: Het huidige tijdstip tonen en automatisch bijwerken.
  • Functie: Een JavaScript-functie die elke seconde het tijdstip in de koptekst bijwerkt.
  • Interactie: De koptekst toont de actuele tijd en wordt continu bijgewerkt.
