Oefening: Wijzigen van Inline CSS Stijlen

In deze oefening leer je hoe je met JavaScript inline CSS-stijlen kunt toepassen op HTML-elementen.

Voorbeeldcode
// Selecteer het element met id "intro"
let introElement = document.getElementById("intro");

// Pas verschillende stijlen toe op het element
introElement.style.color = "blue"; // Tekstkleur wordt blauw
introElement.style.fontSize = "42px"; // Lettergrootte wordt 42px
introElement.style.backgroundColor = "yellow"; // Achtergrondkleur wordt geel

console.log(introElement); // Logt het element naar de console
Welkom!

Hoe werkt deze oefening?

  • Wijziging van tekstkleur: Met introElement.style.color = "blue" verandert de kleur van de tekst naar blauw.
  • Wijziging van lettergrootte: Met introElement.style.fontSize = "42px" zetten we de lettergrootte op 42px.
  • Wijziging van achtergrondkleur: Met introElement.style.backgroundColor = "yellow" veranderen we de achtergrondkleur naar geel.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomsten te volgen.
