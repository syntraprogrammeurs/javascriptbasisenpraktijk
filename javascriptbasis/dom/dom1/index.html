Oefening: Selectie van Elementen met JavaScript

In deze oefening leer je hoe je verschillende HTML-elementen kunt selecteren met JavaScript en de inhoud ervan in de console kunt bekijken.

Voorbeeldcode
// Selecteer het element met id "intro" en log de tekstinhoud
let introElement = document.getElementById("intro");
console.log(introElement.textContent);

// Selecteer alle elementen met class "bericht" en log de tekst van het tweede element
let berichten = document.getElementsByClassName("bericht");
console.log(berichten[1].textContent);

// Selecteer alle 
-elementen en log de tekstinhoud van het eerste 
-element
let paragrafen = document.getElementsByTagName("p");
console.log(paragrafen[0].textContent);

// Selecteer het element met id "container" en log de tekst van het eerste kind
let containerparagraaf = document.querySelector("#container");
console.log(containerparagraaf.children[0].textContent);

// Selecteer alle 
  •  elementen binnen een 
       en log het aantal 
    •  elementen
let items = document.querySelectorAll("ul li");
console.log(items.length);

    Hoe werkt deze oefening?

    • Selectie op ID: Met getElementById("intro") selecteren we het element met de ID intro en loggen de tekstinhoud "Welkom bij de pagina!"
    • Selectie op Class: Met getElementsByClassName("bericht") selecteren we alle elementen met de class bericht en loggen de tekst van het tweede element, "Bericht 2".
    • Selectie op Tagnaam: Met getElementsByTagName("p") selecteren we alle <p> elementen en loggen de tekst van het eerste <p> element, "Dit is paragraaf 1."
    • Selectie op Kinderen: Met querySelector("#container") selecteren we het element met ID container en loggen de tekst van het eerste kindelement, "Dit is een paragraaf binnen de container."
    • Selectie van <li> elementen in een <ul>: Met querySelectorAll("ul li") selecteren we alle <li> elementen in een <ul> en loggen we het aantal elementen, wat hier 3 is.

    Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomst van elke selectie te zien.
