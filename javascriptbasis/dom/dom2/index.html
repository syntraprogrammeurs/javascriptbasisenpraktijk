Oefening: Wijzigen van InnerHTML en Attribute

In deze oefening leer je hoe je de inhoud van een HTML-element en een attribuut (zoals src van een afbeelding) kunt wijzigen met JavaScript.

Voorbeeldcode
// Wijzig de innerHTML van het element met id "intro"
let introElement = document.querySelector("#intro");
introElement.innerHTML = "Dit is een wijziging";

// Wijzig het src-attribuut van de afbeelding met id "logo"
let logo = document.getElementById("logo");
logo.setAttribute("src", "nieuwe_logo.png");
Welkom!

Hoe werkt deze oefening?

  • Wijziging van innerHTML: Met querySelector("#intro") selecteren we het element met de ID intro. Vervolgens wordt de inhoud van dit element gewijzigd naar <strong>Dit is een wijziging</strong>, wat vetgedrukte tekst toevoegt.
  • Wijziging van attribuut: Met getElementById("logo") selecteren we de afbeelding met de ID logo. Vervolgens veranderen we het src-attribuut naar nieuwe_logo.png, zodat een andere afbeelding wordt weergegeven.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomsten te volgen.
