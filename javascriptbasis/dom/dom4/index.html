Oefening: Toevoegen en Verwijderen van Elementen

In deze oefening leer je hoe je een nieuw element toevoegt aan de body en hoe je een specifiek element binnen een container kunt verwijderen.

Voorbeeldcode
// Maak een nieuw 
-element en voeg het toe aan de body
let nieuwParagraaf = document.createElement("p");
nieuwParagraaf.textContent = "Dit is een nieuwe paragraaf";
document.body.appendChild(nieuwParagraaf);

// Selecteer de container en verwijder een specifiek 
-element
let container = document.getElementById("container");
let paragraaf = document.getElementById("paragraaf");
container.removeChild(paragraaf);
Welkom!

Dit is tekst van de container

Hoe werkt deze oefening?

  • Toevoegen van een nieuw element: Met document.createElement("p") maken we een nieuw <p>-element. Vervolgens voegen we dit element toe aan de body met document.body.appendChild(nieuwParagraaf), waardoor de tekst "Dit is een nieuwe paragraaf" verschijnt.
  • Verwijderen van een bestaand element: Met container.removeChild(paragraaf) selecteren we het <p>-element binnen de container met ID paragraaf en verwijderen we het, zodat deze tekst niet meer zichtbaar is.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomsten te volgen.
