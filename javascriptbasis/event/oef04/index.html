Oefening: Verander de Tekst bij Mouseout
Voorbeeldcode
// JavaScript voor mouseout event
document.getElementById("mouseoutText").addEventListener("mouseout", function() {
    console.log("Bedankt voor je bezoek!"); // Logt het bericht naar de console
    document.getElementById("output").textContent = "Bedankt voor je bezoek!";
});
Beweeg de muis van de tekst hieronder of klik op "Run" om het resultaat te zien.

Beweeg de muis van deze tekst

Uitleg

In deze oefening voegen we een mouseout event toe aan een tekstregel. Het doel is om "Bedankt voor je bezoek!" te loggen in de console en weer te geven in het output-vak wanneer de muis van de tekst wordt weggehaald.

  • Doel: Demonstreren hoe je een mouseout event aan een tekst toevoegt met JavaScript.
  • Functie: De functie logt het bericht "Bedankt voor je bezoek!" in de console en toont het bericht in het output-vakje.
  • Run-knop: De "Run"-knop simuleert het mouseout event op de tekst zodat je de uitkomst kunt zien.

Beweeg met de muis van de tekst of gebruik de "Run"-knop om het resultaat te zien.
