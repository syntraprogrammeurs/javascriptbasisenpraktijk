Oefening: Aantal Klikken bijhouden
Voorbeeldcode
// JavaScript voor klikken tellen
let klikAantal = 0; // Start met 0 klikken

// Event listener voor de knop
document.getElementById("klikKnop").addEventListener("click", function() {
    klikAantal++; // Verhoog het aantal klikken
    console.log("Knop geklikt: " + klikAantal + " keer"); // Log het aantal klikken
    document.getElementById("output").textContent = "Knop geklikt: " + klikAantal + " keer";
});
Klik op de knop om het aantal te zien.

Uitleg

In deze oefening leer je hoe je een click event gebruikt om bij te houden hoe vaak er op een knop wordt geklikt.

  • Doel: Het aantal klikken bijhouden en tonen in de console en op de pagina.
  • Functie: Telkens wanneer je op de knop "Klik mij!" klikt, wordt het aantal klikken bijgehouden en bijgewerkt in de console en in de outputbox.
  • Run-knop: Klik op de knop om het aantal klikken direct te verhogen en te tonen.
