Oefening: Gebruikersinteractie met JavaScript

In deze oefening leer je hoe je JavaScript kunt gebruiken om reacties te triggeren bij gebruikersinteracties zoals klikken, formulierinzending en toetsenbordinvoer.

Voorbeeldcode
let knop = document.getElementById("klikmij");
knop.addEventListener("click", function() {
    alert("Je hebt op mij geklikt!");
    document.body.style.backgroundColor = "blue";
    knop.textContent = "Ik ben aangeklikt";
});

let form = document.getElementById("mijnForm");
form.addEventListener("submit", function(event) {
    event.preventDefault();
    let naam = document.getElementById("naamInput").value;
    console.log("Naam:", naam);
    document.getElementById("naamInput").value = "";
});

document.addEventListener("keydown", function(event) {
    console.log("Toets ingedrukt:", event.key);
});

Hoe werkt deze oefening?

  • Knop Click Event: Als je op de knop "Klik Mij" klikt, verschijnt er een alert, verandert de achtergrondkleur naar blauw, en verandert de tekst van de knop.
  • Formulierinzending: Wanneer je het formulier indient, wordt de pagina niet herladen. De ingevoerde naam wordt in de console weergegeven en het invoerveld wordt geleegd.
  • Toetsenbordinvoer: Bij elke toetsaanslag wordt de naam van de ingedrukte toets in de console weergegeven.

Druk op F12 of gebruik Ctrl + Shift + J (Windows) of Cmd + Option + J (Mac) om de console te openen en de uitkomst van elke interactie te zien.
