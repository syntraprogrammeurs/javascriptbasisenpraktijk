Oefening: Toon de Geklikte Coördinaten in de Console
Voorbeeldcode
// JavaScript voor klikcoördinaten
document.addEventListener("click", function(event) {
    const x = event.clientX; // x-coördinaat van de klik
    const y = event.clientY; // y-coördinaat van de klik
    console.log(`Klikpositie: x=${x}, y=${y}`);
    document.getElementById("output").textContent = `Klikpositie: x=${x}, y=${y}`;
});
Klik ergens op de pagina om de coördinaten te zien of klik op "Run".

Uitleg

In deze oefening voegen we een click event toe aan de hele pagina. Dit event logt de x- en y-coördinaten van de klikpositie in de console en toont deze in het output-vak.

  • Doel: Leren hoe je een click event op de pagina gebruikt om coördinaten vast te leggen.
  • Functie: De functie leest de klikpositie van de muis en logt deze in de console. De coördinaten worden tevens getoond in het output-vak.
  • Run-knop: De "Run"-knop simuleert een klik in het midden van de pagina om de coördinaten te bekijken.

Klik ergens op de pagina of gebruik de "Run"-knop om de klikpositie te zien.
