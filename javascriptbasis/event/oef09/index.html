Oefening: Inhoud van een invoerveld tonen tijdens typen
Voorbeeldcode
// JavaScript voor het tonen van invoer tijdens typen
document.getElementById("invoerVeld").addEventListener("input", function(event) {
    // Haal de huidige waarde uit het invoerveld op
    const huidigeInvoer = event.target.value;
    console.log("Invoer: " + huidigeInvoer); // Log de huidige invoer
    document.getElementById("output").textContent = "Invoer: " + huidigeInvoer; // Toon de invoer in het outputvak
});
Invoer verschijnt hier...

Uitleg

In deze oefening leer je hoe je een input event gebruikt om de invoer van een gebruiker realtime te tonen.

  • Doel: Het invoerveld bijhouden en tonen in de console en op de pagina terwijl de gebruiker typt.
  • Functie: Bij elke invoer toont de console en het outputvak de huidige tekst.
  • Interactie: Typ in het veld om te zien hoe de tekst realtime wordt weergegeven.
