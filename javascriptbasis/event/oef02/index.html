Oefening: Verander de Achtergrondkleur met een Double-click Event
Voorbeeldcode
// JavaScript functie om achtergrondkleur te veranderen bij double-click
document.getElementById("colorBox").addEventListener("dblclick", function() {
    document.getElementById("colorBox").style.backgroundColor = "lightblue"; // Verander de achtergrondkleur
    console.log("Achtergrondkleur veranderd!"); // Logt het bericht naar de console
    document.getElementById("output").textContent = "Achtergrondkleur veranderd!";
});
Klik op "Run" om het resultaat te zien.
Double-click op dit vlak om de achtergrondkleur te veranderen

Uitleg

In deze oefening voegen we een double-click event toe aan een div. Het doel is om de achtergrondkleur van de div te veranderen naar lichtblauw en een bericht in de console te loggen wanneer de gebruiker dubbelklikt.

  • Doel: Demonstreren hoe je een double-click event toevoegt aan een div met JavaScript.
  • Functie: De functie document.getElementById("colorBox").style.backgroundColor = "lightblue" verandert de kleur en logt "Achtergrondkleur veranderd!" in de console en in het output-vakje.
  • Run-knop: De "Run"-knop simuleert een double-click op de div zodat je de uitkomst kunt zien.

Double-click op de div of gebruik de "Run"-knop om het resultaat te zien.
