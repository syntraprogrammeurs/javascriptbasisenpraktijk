Click Event toevoegen aan een Knop

Terug naar Home
Oefening: Voeg een Click Event toe aan een Knop om een Bericht weer te geven
Voorbeeldcode
// JavaScript functie om een bericht in de console weer te geven bij een klik
document.getElementById("clickButton").addEventListener("click", function() {
    console.log("Knop is geklikt!"); // Logt het bericht naar de console
});
Klik op "Run" om het resultaat te zien.

Uitleg

In deze oefening voegen we een click event toe aan een knop met JavaScript. Het doel is om een bericht naar de console te loggen wanneer de gebruiker op de knop klikt.

  • Doel: Demonstreren hoe je een click event toevoegt aan een knop met JavaScript.
  • Functie: De functie console.log("Knop is geklikt!") toont het bericht in de console en in het output-vakje.
  • Run-knop: De "Run"-knop simuleert een klik op de knop "Klik Mij" zodat je de uitkomst kunt zien.

Klik op de knop "Klik Mij" of gebruik de "Run"-knop om het resultaat te zien.
