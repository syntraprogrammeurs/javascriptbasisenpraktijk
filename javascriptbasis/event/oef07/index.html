Oefening: Actie bij Scrollen
Voorbeeldcode
// JavaScript voor scroll event
window.addEventListener("scroll", function() {
    let scrollPos = window.scrollY; // Verkrijg huidige scrollpositie
    console.log("Scrollniveau: " + scrollPos + "px"); // Log de scrollpositie in px
    document.getElementById("output").textContent = "Scrollniveau: " + scrollPos + "px";
});
Scroll omlaag om het scrollniveau te zien of klik op "Run".

Scroll verder naar beneden om de output van de scrollpositie in de console te zien en in het outputvak hieronder.

Uitleg

In deze oefening voegen we een scroll event toe aan de pagina. Wanneer je naar beneden scrolt, logt de console het huidige scrollniveau in pixels.

  • Doel: Leren hoe je scroll events kunt gebruiken om acties uit te voeren bij scrollen.
  • Functie: Bij elk scrollniveau toont de console de actuele positie in pixels.
  • Run-knop: Klik op de "Run" knop om de huidige scrollpositie in de outputbox te zien.
