Oefening: Toon een Bericht bij Mouseover
Voorbeeldcode
// JavaScript voor mouseover event
document.getElementById("hoverText").addEventListener("mouseover", function() {
    console.log("Je beweegt over de tekst!"); // Logt het bericht naar de console
    document.getElementById("output").textContent = "Je beweegt over de tekst!";
});
Hover over de tekst hieronder of klik op "Run" om het resultaat te zien.

Beweeg met je muis over deze tekst

Uitleg

In deze oefening voegen we een mouseover event toe aan een tekstregel. Het doel is om "Je beweegt over de tekst!" te loggen in de console en weer te geven in het output-vak wanneer de muis over de tekst beweegt.

  • Doel: Demonstreren hoe je een mouseover event aan een tekst toevoegt met JavaScript.
  • Functie: De functie logt het bericht "Je beweegt over de tekst!" in de console en toont het bericht in het output-vakje.
  • Run-knop: De "Run"-knop simuleert het mouseover event op de tekst zodat je de uitkomst kunt zien.

Beweeg met de muis over de tekst of gebruik de "Run"-knop om het resultaat te zien.
